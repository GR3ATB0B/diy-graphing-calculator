## Total Keys: ~50 physical keys

## Key UX Notes:
- 2nd is YELLOW — secondary functions printed in yellow above keys
- ALPHA is GREEN — letters printed in green above keys
- Graphing row (Y=, WINDOW, ZOOM, TRACE, GRAPH) is the TOP ROW — instant access
- Arrow keys in a circular cluster, upper right
- ENTER is the biggest key (bottom right)
- Number pad is standard 7-8-9 / 4-5-6 / 1-2-3 / 0 layout
- Operations (÷ × − +) run down the right column
- CLEAR is on its own, easy to reach
